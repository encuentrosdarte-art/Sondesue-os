# Son de Sueños 🎶

Sitio web para la venta de entradas de conciertos y eventos musicales **Son de Sueños**.

## Descripción

Página web estática con un diseño elegante (paleta dorada y morada sobre fondo oscuro) que incluye:

- **Navbar** con logo y navegación.
- **Hero** con el logo como fondo (opacidad 13%, con desenfoque y saturación).
- **Sección de Eventos** con tarjetas de conciertos y modal de compra de entradas.
- **Sección de Géneros** musicales.
- **Sección Nosotros** (About Us).
- **Formulario de Contacto**.
- **Footer** con información legal.

## Estructura del proyecto

```
.
├── index.html        # Página principal
├── css/
│   └── styles.css    # Estilos
├── js/
│   └── script.js     # Lógica (eventos, géneros, modal)
└── images/
    └── logo.jpeg     # Logo de Son de Sueños
```

## Uso

Abre `index.html` en cualquier navegador moderno. No requiere dependencias ni servidor.
